# 一松プロジェクト (pj_Hitotsumatsu)

## 概要
千葉県長生郡長生村一松戊に位置する、競売で取得した大規模土地（約1,563坪）の売却・有効活用に向けたプロジェクトです。

## 基本情報
- **所在地**: 千葉県長生郡長生村一松戊3451-35 他
- **面積**: 5,168m² (約1,563坪)
- **取得価格**: 1,666万円
- **取得経緯**: グループ会社が約4年前（2022年頃）に競売により取得
- **目的**: 売却、または開発・有効活用の検討
