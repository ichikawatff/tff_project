# マーケティング管理

株式会社東京福祉不動産および「HIDAKA STAY VILLA」の集客・ブランド構築に関する管理ドキュメントです。

## 概要

本セクションでは、民泊事業（HIDAKA STAY VILLA）を中心とした集客戦略、SNS運用、ブランド素材の管理を行います。

## ドキュメント構成

- [SNS運用ガイドライン](sns_guideline.md)
  - Instagram, YouTube, TikTok, Googleビジネスプロフィールの運用ルールとガイドライン。
- [ブランド素材 (brand/)](brand/README.md)
  - ロゴ、公式写真、カラーパレットなどの素材管理。

## 運用方針

1. **グローバルターゲット**: 世界中から訪れるゲストを対象とし、英語を主軸とした情報発信を行う。
2. **体験価値の向上**: 単なる宿泊場所の紹介ではなく、柴又という地域の魅力や、チェックインの利便性（安心感）を伝える。
3. **デジタル活用**: SNSとGoogleビジネスプロフィールを連携させ、予約前から滞在中までシームレスなサポートを提供する。
