# SNS運用ガイドライン

## 1. 運用の目的
- **ブランド認知向上**: 「HIDAKA STAY VILLA (HSV)」の名称と柴又の魅力を世界に広める。
- **ゲストの利便性向上**: チェックイン方法などの実用的な動画を提供し、滞在の不安を解消する。
- **集客の最大化**: OTA（Airbnb, Booking.com）への誘導および直接予約の獲得。

## 2. 対象プラットフォームと役割
| プラットフォーム | 主な役割 | ターゲット |
| :--- | :--- | :--- |
| **Instagram (Reels)** | 視覚的なブランディング・クイックガイド | 旅行を計画中の20-40代 |
| **YouTube (Shorts)** | 詳細な施設案内・ハウツー動画 | 予約確定後のゲスト・じっくり探したい層 |
| **TikTok** | トレンドに合わせた認知拡大 | Z世代・ミレニアル世代の旅行者 |
| **Googleビジネスプロフィール** | 到着時の実用サポート（解錠方法等） | 現地に到着した・または直前のゲスト |

## 3. コンテンツ指針
- **実用系 (Utility)**: 玄関の解錠方法、Wi-Fi接続、ゴミの捨て方、周辺の駅からの道順。
- **体験系 (Vibe)**: 柴又の街並み（帝釈天、参道）、物件内の和の雰囲気、朝の風景。
- **情報系 (Info)**: おすすめの飲食店、周辺の隠れスポット、季節のイベント。

## 4. 投稿の基本ルール
- **言語**: 原則として**英語メイン**（世界ターゲット）。必要に応じて日本語を併記。
- **トーン**: 親しみやすく、かつプロフェッショナルで安心感を与える。
- **ハッシュタグ**: 以下のタグを組み合わせて使用。
  - `#hidakastayvilla` `#shibamata` `#tokyostay` `#visittokyo` `#japantrip` `#traditionaljapan` `#airbnbjapan`
