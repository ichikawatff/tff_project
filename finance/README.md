# 財務・会計管理

## 基本方針
会計処理は **マネーフォワード クラウド会計 (MFクラウド)** をメインに使用します。
AIとの連携をスムーズにするため、MCP (Model Context Protocol) を活用してデータを取得・分析します。

## フォルダ構成
- `mf_cloud/`: MCP連携用のスクリプトや分析結果
- `expenses/`: 日々の経費入力、領収書管理など

## 決済手段
- **銀行口座**: 楽天銀行、東京東信用金庫（詳細は `confidential/bank_accounts.md`）
- **クレジットカード**: 用途別に3枚（詳細は `confidential/credit_cards.md`）
    - 会社全般: 末尾 6603
    - 柴又用: 末尾 1649
    - 飯岡用: 末尾 5577
